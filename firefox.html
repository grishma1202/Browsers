Firefox

A short brief information on Firefox

  1. Firefox ia a freely available, open source web browser by Mozilla Application Suite.
  2. It is the second most widely used browser after Chrome.
  3. It uses Gecko layout engine, which is known to implement most current web standards.
  4. The code is based on Mozilla Code and is available for Windows 98 , Mac OS , Linux , etc

Features of Firefox are as follows:
1. Imporved look and feel.
2. Trace Monkey Java Script Engine.
3. Allows private browsing.
4. Pop-up Blocker.
5. Location Aware Browsing.
6. Improved Text Rendering.
7. Live Bookmarking.
