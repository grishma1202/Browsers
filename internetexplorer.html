Internet Explorer

A short brief information on Internet Explorer

  1. It is commonly abbreviated as IE is a graphical web browser.
  2. It is developed by Microsoft and is included as a part of Windows Operating System since 1995.
  3. This browser finds an upgrade with every upgraded windows version.
  4. Internet Explorer 11 features in windows 8.1 and was released on October 17,2013.

Features of Internet Explorer are as follows:
1. Private Browsing
2. Save Passwords
3. Reading Mode
4. Improved Tabs and Browsing
5. High Savings Mode
6. Better File Downloads
