Opera

A short brief information on Opera.

  1. Opera is a browser and Internet suite developed by Opera software.
  2. It is one of the fastest and smallest browsers.
  3. It is also compatible with operating systems like Microsoft Windows , Mac and Linux
  4. The browser also handles common internet related functions such as displaying web pages , online chatting , etc.

Features of Opera are as follows:
1. Tabbed Browsing
2. Page Zooming
3. Mouse Gestures
4. Download Manager
5. Built-in Phishing
6. Malware Protection
7. Strong Encryption
8. HTTP cookies
